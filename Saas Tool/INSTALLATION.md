# Installation Guide

## Quick Start

1. **Download/Clone the Extension**
   - Make sure all files are in the `MA-LALYTICS` directory

2. **Open Chrome Extensions Page**
   - Navigate to `chrome://extensions/` in your Chrome browser
   - Or go to: Chrome Menu → More Tools → Extensions

3. **Enable Developer Mode**
   - Toggle the "Developer mode" switch in the top-right corner

4. **Load the Extension**
   - Click "Load unpacked" button
   - Select the `MA-LALYTICS` directory
   - The extension should now appear in your extensions list

5. **Add Icons (Optional)**
   - The extension will work without icons, but you can add custom icons:
   - Place `icon16.png`, `icon48.png`, and `icon128.png` in the `icons/` folder
   - See `icons/README.md` for more details

6. **Start Using**
   - Navigate to Gmail (https://mail.google.com)
   - The extension will automatically start tracking contacts
   - Click the extension icon in Chrome toolbar to access settings
   - Use the floating widget (bottom-right) to open the dashboard

## First Time Setup

1. **Configure Settings**
   - Click the extension icon in Chrome toolbar
   - Adjust settings:
     - Dark Mode: Toggle theme
     - Auto Approve: Skip approval prompts
     - Enable Sidebar: Show/hide sidebar in Gmail
     - Reminder Days: Set follow-up reminder interval

2. **Start Using Gmail**
   - The extension automatically scans emails as you send/receive them
   - Approval panels will appear when new contacts are detected
   - Use the sidebar to view your contact dashboard

## Features Overview

- **Automatic Contact Detection**: Works in the background as you use Gmail
- **Approval Workflow**: Review contacts before they're added
- **Dashboard**: View stats and manage contacts via sidebar
- **CSV Export**: One-click export for CRM import
- **Reminders**: Automatic follow-up scheduling
- **Bulk Actions**: Approve multiple contacts at once

## Troubleshooting

**Extension not working?**
- Make sure you're on Gmail (mail.google.com)
- Refresh the Gmail page
- Check that the extension is enabled in `chrome://extensions/`

**Contacts not being detected?**
- The extension scans emails as you interact with them
- Try sending or receiving a new email
- Check browser console for any errors (F12 → Console)

**Sidebar not showing?**
- Click the floating widget (bottom-right corner)
- Or use the "View Dashboard" button in the popup
- Make sure "Enable Sidebar" is checked in settings

## Privacy Note

All data is stored locally in your browser. No information is sent to external servers. Your contacts remain private and secure.
