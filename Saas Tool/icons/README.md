# Extension Icons

Please add the following icon files to this directory:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

You can create these icons using any image editor. The icons should represent an email/contact management theme. For a quick start, you can:

1. Use an online icon generator
2. Create simple icons with a mail/contact symbol
3. Use free icon resources like:
   - Flaticon
   - Icons8
   - Material Icons

The extension will work without these icons, but Chrome will show a default placeholder icon.
