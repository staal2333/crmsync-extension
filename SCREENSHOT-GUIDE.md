## üéØ **HOW TO TAKE SCREENSHOTS:**

### **Method 1: Chrome DevTools (Recommended)**
1. Open Chrome
2. Press `F12` to open DevTools
3. Click the device toolbar icon (or Ctrl+Shift+M)
4. Set responsive dimensions to `1280 x 800`
5. Navigate to the page you want to capture
6. Press `Ctrl+Shift+P`, type "Capture screenshot", select "Capture screenshot"
7. Save as PNG

### **Method 2: Windows Snipping Tool**
1. Open Snipping Tool
2. Set to rectangular snip
3. Capture the window at 1280x800
4. Save as PNG
